OSCAAR - An Open Source Photometry Code For Everyone

Check this project out on GitHub!

The Open Source differential photometry Code for Amateur Astronomical Research, or OSCAAR is a project I started as an undergraduate at the University of Maryland. I wanted to study extrasolar planets using the small telescopes at the University of Maryland Observatory, so I wrote up a package of Python tools for differential photometry with a specific focus on applying it to exoplanet transit photometry with small observatories. After I started regularly producing light curves with the code I formed teams to help develop OSCAAR as it is today, with a friendly graphical user interface and lots of documentation.

OSCAAR is intended to be a Python package for undergraduates or advanced amateurs who have access to basic observing facilities and want to get their hands dirty with transit photometry. It is meant to be an easy to use, graphical package for everything you need from planning which transits to observe, to producing light curves from raw FITS data, to fitting analytical models to those light curves.

Since I worked on OSCAAR a bit while I was at NASA's Goddard Space Flight Center with support from Dr. Avi Mandell (GSFC), NASA wrote up a great press release on the project that got some fun traction in the media.

TwitterBots for Science

Sometimes I make automated Twitter accounts that tweet about transient scientific events, controlled by Python scripts. Check them out and be sure to follow my human account @brettmor!

@transitingnow

Whenever a planet in another solar system passes in front of its host star from our perspective on the Earth (i.e. an exoplanet transit), @transitingnow tweets announcing the transit and sharing fun facts about the planet.

@farawayplanets

Light takes time to travel, so observers on other planets would see Earth as it appeared at a time in the past. Ever wonder what historical events observers on known exoplanets would see if they looked up at Earth right now? @farawayplanets tells you!

Read on for more details...

@planettvguide

Light takes time to travel, so anyone in the universe with an old TV and sensitive enough antennae can pick up Earth's television broadcast, from the days when TV was transmitted over the air. This TwitterBot tweets during the arrivals of premiere broadcasts of American television shows at known exoplanets within 100 light years.
